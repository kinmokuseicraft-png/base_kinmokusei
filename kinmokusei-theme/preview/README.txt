プレビューの開き方
================

【方法1】ブラウザで直接開く
  preview/index.html をダブルクリック、またはブラウザにドラッグ＆ドロップ。
  （main.css と js は同じフォルダにコピー済みなので相対パスで読み込みます）

【方法2】ローカルサーバーで開く（推奨・画像も表示される）
  ターミナルで以下を実行してから、ブラウザで http://localhost:8080/preview/ を開く。

    cd c:\Users\tsuba\Desktop\base_kinmokusei\kinmokusei-theme
    python -m http.server 8080

  止めるときはターミナルで Ctrl+C。

確認できる挙動
--------------
・ローディング：キツネ/トカゲ/リス/オオカミのいずれかが表示 → 約1.2秒でフェードアウト
・メインがフェードイン
・スクロールでヒーロー・商品画像がモノクロ → カラーに変化
